Ideal Job Comparison

Within the group there are only three different ideal jobs chosen, software developer, full stack developer, and database specialist. Software development and full stack development are very similar as they are both development roles revolving around creating applications with programming languages, however; in a software development role you will be creating, testing, and maintaining an applications back-end which will contain logic and databases, while a full stack developer will primarily by creating front-end UI and back-end logic and databases for web apps or websites. Database specialists design and install database systems for their employers or clients and they can also be a part of the development process of applications which require complex databases to retrieve or display data.

Comparison and Similarities
Inherently, all three of the ideal jobs listed in the comparison table have similar requirements and skills needed as they all revolve around the development aspect of information technologies. Where many differences lie is in the duties and tasks for each role.

When comparing a software developer to a full stack developer the differences between these roles are the tools they are using to develop, the scope of development, and the product of development. A software developer will mostly be using programming languages such as python, objective-c, java, and swift which are used to develop native applications, whereas a full stack developer will be using front-end languages such as HTML, CSS, and JavaScript as well as the back-end languages which software developers use such as python, objective-c, java, and swift. Both the software developer and full stack developer rolls will be using IDEs to create their applications and similar collaborative tools when working in a team like GitHub and MS Teams. As for skills, both the software developer and the full stack developer require similar, if not the same skill set. The only difference would be that because of the nature of front-end development, full stack developers will need a form of creativity when developing their websites or webapps.

Comparing a database specialist to a software developer and full stack developer the tools used are different. Rather than using an IDE, database specialists use some different tools such as Microsoft SQL which is a powerful tool used to create, maintain, and access data. Database specialists will also require skills in SQL Scripting to input, retrieve and maintain data within the databases they manage. As for skills required database specialists differ from software developers and full stack developers as a core skill being incredible attention to detail and patience. The duties and tasks involved with being a database specialist differ from those of a software developer or full stack developer as they are not working with applications. Database specialists are to create, maintain, and perform data entry to databases as well as provide security and support the users of the database.

In conclusion, all three of the ideal jobs chosen by our group are very similar in nature with only slight differences in the requirements and skills needed. All these jobs require bachelor’s degrees in IT or a related field, experience with coding, and experience using the relevant applications and tools used in the job. For the skills needed all the jobs require a high attention to detail, the ability to work independently and also collaboratively, and finally incredible problem solving skills, where the differences lie is the duties and tasks involved in the job. Software developers create applications, full stack developers create webapps and websites front-end and back-end, and database specialists create intricate databases.
